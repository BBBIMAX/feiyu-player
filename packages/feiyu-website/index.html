常见问题

    • 可以投屏吗?

      暂不支持，推荐使用夸克浏览器，支持投屏

    • 可以下载视频吗？

      暂不支持，推荐使用夸克浏览器，可离线缓存视频

    • 为什么搜索不到结果？

      请在设置中检查你的请求代理和资源站是否可用

    • 为什么提示分享失败？

      分享使用去中心化存储，需要配置正确的 API Key

    • 有客户端吗？

      暂无，飞鱼支持PWA，可离线使用
